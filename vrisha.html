About

vrisha (Autonomous Mobile Robot)

"vrisha" is an Industrial AMR (Autonomous Mobile Robot) controlled by using PLC and designed to autonomously navigate, map, and interact with its environment in real-time. The robot integrates state-of-the-art hardware and software components to achieve high-performance, energy-efficient mobility and sensing capabilities.

This project is in development Stage and its a virtual simulation robot and a physical Robot.
